Politique de confidentialité

Dernière mise à jour : janvier 2025

1. Informations que nous collectons

Prep DELE C2 collecte les informations minimales nécessaires pour fournir le service :

  • Données de compte : Si vous créez un compte, nous stockons votre adresse e-mail pour l'authentification.
  • Données de progression : Votre progression dans les examens pratiques est stockée localement sur votre appareil et, si vous avez un compte, elle est synchronisée de manière sécurisée.
  • Données d'utilisation anonymes : Nous collectons des données analytiques anonymes (pages visitées, fonctionnalités utilisées) via Google Analytics pour améliorer l'application.

2. Informations que nous NE collectons PAS

  • Nous ne vendons pas vos données personnelles à des tiers.
  • Nous ne collectons pas de données de localisation.
  • Nous n'accédons pas à vos contacts, photos ni à d'autres données de l'appareil.

3. Abonnements et paiements

Les paiements sont traités exclusivement par Apple (App Store). Nous n'avons pas accès à vos informations de paiement (carte de crédit, etc.). Apple gère toutes les transactions selon sa propre politique de confidentialité.

4. Stockage des données

Vos données de progression sont principalement stockées sur votre appareil. Si vous créez un compte, les données sont synchronisées via des services sécurisés dans le cloud avec un chiffrement en transit et au repos.

5. Suppression des données

Vous pouvez supprimer votre compte et toutes les données associées à tout moment depuis l'application (Compte → Supprimer le compte). Vous pouvez également demander la suppression en envoyant un e-mail à hello@prepdele.com.

6. Services tiers

  • Google Analytics : Pour l'analyse d'utilisation anonyme.
  • Firebase : Pour l'authentification et le stockage des données.
  • RevenueCat : Pour la gestion des abonnements.

7. Mineurs

L'application n'est pas destinée aux moins de 13 ans. Nous ne collectons pas intentionnellement d'informations sur les moins de 13 ans.

8. Changements dans cette politique

Nous pouvons mettre à jour cette politique occasionnellement. Nous vous informerons des changements significatifs via l'application ou par e-mail.

9. Contact

Si vous avez des questions concernant cette politique de confidentialité, écrivez-nous à hello@prepdele.com.
